Logic & Conditional Statements
If-Then, Converse, Inverse, Contrapositive

Quick Reference

Conditional Statement

If p, then q (p → q)

  • Hypothesis: The "if" part (p)
  • Conclusion: The "then" part (q)
  • Example: If it rains, then the ground is wet

Related Statements

  • Converse: If q, then p (swap)
  • Inverse: If not p, then not q (negate)
  • Contrapositive: If not q, then not p (swap & negate)
Key Facts:
  • A statement and its contrapositive are logically equivalent (both true or both false)
  • The converse and inverse are logically equivalent to each other
  • Biconditional (p ↔ q): True when both p → q AND q → p are true

Quick Reference

Proof Types

  • Two-Column: Statements | Reasons
  • Paragraph: Written explanation with logical flow
  • Flowchart: Visual diagram with arrows
  • Indirect (Contradiction): Assume opposite, find contradiction

Common Reasons

  • Given
  • Definition of (congruence, midpoint, etc.)
  • Reflexive, Symmetric, Transitive Properties
  • Substitution Property
  • Vertical Angles Theorem
Properties of Equality:
  • Reflexive: a = a (anything equals itself)
  • Symmetric: If a = b, then b = a
  • Transitive: If a = b and b = c, then a = c

Transformations
Rigid & Non-Rigid Transformations

Quick Reference

Rigid Transformations (Preserve Distance)

  • Translation: (x, y) → (x + a, y + b)
  • Reflection over x-axis: (x, y) → (x, -y)
  • Reflection over y-axis: (x, y) → (-x, y)
  • Rotation 90° CCW: (x, y) → (-y, x)
  • Rotation 180°: (x, y) → (-x, -y)

Non-Rigid Transformations

  • Dilation: (x, y) → (kx, ky)
  • k > 1: Enlargement
  • 0 < k < 1: Reduction
  • Preserves angle measures, NOT distances
  • Creates similar (not congruent) figures
Key Fact: Rigid transformations preserve both distance AND angle measure. Figures are CONGRUENT after rigid transformations.

Triangle Theorems & Congruence
Triangle Properties, SSS, SAS, ASA, AAS, HL

Quick Reference

SSS
3 sides ≅
SAS
2 sides + included angle
ASA
2 angles + included side
AAS
2 angles + non-included side
HL
Right triangles: Hyp + Leg
AA (Similarity)
2 angles ≅ → Similar
Triangle Theorems:
  • Interior angles sum to 180°
  • Exterior angle = sum of two remote interior angles
  • Triangle Inequality: Any two sides sum > third side
  • Base angles of isosceles triangles are congruent

Coordinate Geometry
Distance, Midpoint, Slope, and Classification

Quick Reference

Distance Formula
d = √[(x2-x1)² + (y2-y1)²]
Midpoint Formula
M = ((x1+x2)/2, (y1+y2)/2)
Slope Formula
m = (y2-y1)/(x2-x1)
Parallel Lines
Same slope: m1 = m2
Perpendicular Lines
m1 · m2 = -1
Slopes
Negative reciprocals
